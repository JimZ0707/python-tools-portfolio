## 🧩 Daftar Tools

### 1. File Renamer
Script ini membantu mengganti nama banyak file sekaligus secara otomatis.

**Fitur:**
- Ganti nama file berurutan
- Deteksi ekstensi otomatis
- Aman (tidak menghapus file)

**Cara Pakai:**
```bash
python file_renamer.py
```

**Contoh Output:**
```
file_1.txt
file_2.txt
file_3.txt
```
